# Ship-Log-Database-
Personal database files for a ship log database. Stack dependencies include MySQL for database handling and Docker for automation. Jupiter used for mapping.  Data scraped from the Internet Archive cataloging Spanish and Scottish ships from the 1400s. 

The findings of the data scraping are placed on a map, and a tracking line is enabled to showcase the timeline of the ship's journey. 

This is an exciting project that helps compare ships mentioned in fables with actual evidence.
